NEWS API Service

### Perquisites ###

* Nodejs
* Git

### How do I get set up? ###

* After Cloning the repo
* Go to root of the project in terminal or in cmd prompt
* Run npm i
* Run npm run dev (for development environment)
* Server will be running on port 3000 by default
